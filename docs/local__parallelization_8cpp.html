Finds the best path with the minimum cost from a set of possible paths.

This function iterates through all possible paths and calculates their respective costs using the provided distances. It then selects the path with the minimum cost.

Parameters
possiblePathsA vector of vectors where each inner vector represents a possible path as a sequence of nodes.
distancesA map that holds the distances between pairs of nodes.
costA reference to an integer where the cost of the best path will be stored.
Returns
A vector of integers representing the nodes in the best path with the minimum cost.

Performs a Nearest Neighbor search for the Traveling Salesman Problem (TSP).

This function constructs a path by visiting the nearest unvisited node and returns to the starting node when necessary, considering the capacity constraints of the nodes.

Parameters
distancesA map where keys are pairs of node indices representing edges and values are the corresponding distances.
nodesA map where keys are node indices and values are the capacities of the nodes.
costAn integer reference that will be set to the total cost of the path found.
maxCapacityThe maximum capacity constraint for the nodes in the path.
Returns
A vector of integers representing the path starting and ending at node 0.
