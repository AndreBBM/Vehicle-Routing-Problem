Finds a path using the nearest neighbor heuristic.

This function finds a path starting from the initial node and repeatedly visits the nearest unvisited node until all nodes are visited. It then returns to the starting node. The cost is calculated based on the distances.

Parameters
distancesA map that holds the distances between pairs of nodes.
nodesA map of nodes with their capacities.
costA reference to an integer where the cost of the path will be stored.
maxCapacityThe maximum capacity allowed for the path.
Returns
